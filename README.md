## Restaurant

## Description:

 Basic Restaurant website to practice bundling with webpack and using javascript modules.

## Built With

## How to Run the App Locally
```
   git clone https://github.com/MarvellousUbani/restaurant
   cd restaurant
   open dist/index.html file

```
